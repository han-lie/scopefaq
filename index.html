Questions about Getting Started

As of 7/1/2025 we no longer enter monthly Beneficiaries at the Project level in the PDT. We now enter Activity-level data at the end of each Activity in Scope Activity app.
Americares now requires estimates on unique participants (Reach), so we use Activity-level location and demographics to estimate how many Beneficiaries may be in multiple Activities.
Activities should have been recorded in your TA Project's PDW. Each has a group of participants, a specific location, and a specified end date. In addition, if you have a Project with multiple Cash Grants, like “Roadmap,” please enter each Grant as a single Activity.
Activity data should be entered when each Activity has ended.
